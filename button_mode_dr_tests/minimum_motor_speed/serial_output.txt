Example serial output — minimum_motor_speed
Board: Arduino Giga R1 @ 115200 baud
Goal: find the lowest Motoron speed that reliably starts each direction from rest.
The motors do not move below ~175; 200 starts every time, so MIN_FORWARD_SPEED=200.

Minimum usable motor speed test
Commands:
  + / -  speed up/down
  f      drive both sides forward
  b      drive both sides backward
  l      left side only
  r      right side only
  s      stop
Tune MIN_SPEED to the lowest speed that starts reliably every time.

testSpeed=300 step=25
