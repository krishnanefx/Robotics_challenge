Commands:
  g      go / start line following
  s      stop
  + / -  base speed up/down
  ] / [  turn gain up/down
  p      toggle sensor/motor printing
  x      flip correction direction if steering is backwards
  k      auto-calibrate by sweeping left/right
  y / h  raw threshold up/down (used before calibration)
  l      test left intersection turn
  r      test right intersection turn
  c      print status
  ?      print this help

If it does not turn enough: increase turnGain with ].
If it turns too slowly: increase baseSpeed with + and/or turnGain with ].
If it drives away from the line: press x.
