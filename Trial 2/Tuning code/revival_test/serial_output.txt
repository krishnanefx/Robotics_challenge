Example serial output — revival_test
Board: Arduino Giga R1 @ 115200 baud
Goal: drive toward a stranded robot, slow inside the approach distance, crawl until
a bumper touches, then smoothly stop. LED: green on bumper contact, red otherwise.

Revival standalone test
Commands:
  g      run revival approach/crawl sequence
  s      stop immediately
  p      print distance, bumper, speed status
  + / -  approach distance up/down by 5 cm
  ] / [  crawl speed up/down by 25
  } / {  approach speed up/down by 25
  > / <  ramp step up/down by 5 speed units
  ?      print help

Expected behavior:
  Red LED on when bumpers are released.
  Green LED on while bumper 22 or 33 is pressed.
  Robot drives fast until front distance <= threshold, then crawls at 200.
  It ramps speed changes smoothly instead of instant jumps.
  On bumper contact, it smoothly stops all Motoron channels and stays stopped.

approachDistanceCm=40 approachSpeed=300 crawlSpeed=150 rampStep=20/20ms currentDriveSpeed=0 front=118 bumper1=released bumper2=released
